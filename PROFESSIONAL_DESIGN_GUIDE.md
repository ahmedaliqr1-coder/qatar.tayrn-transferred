## نظرة عامة على التصميم

تم إعادة تصميم قسم مميزات البطاقة ليطابق معايير التصميم الاحترافي العالمية، مع الحفاظ على الهوية البصرية للبنك.

---

## مكونات التصميم

### 1. شبكة المميزات الرئيسية (2×2 Grid)

#### الخصائص:
- **التخطيط**: شبكة من 4 بطاقات (2 أعمدة × 2 صفوف)
- **الحجم**: 140px ارتفاع لكل بطاقة
- **الفجوة بين البطاقات**: 12px
- **الحدود**: 12px border-radius

#### المكونات الداخلية:

**أ) الصورة الخلفية:**
```
- المواقف: صورة موقف سيارات احترافي
- الطيران: صورة طائرة في السماء
- الفنادق: صورة غرفة فندق فاخرة
- السيارات: صورة سيارة حديثة
```

**ب) الـ Overlay (الطبقة العلوية):**
- خلفية شفافة سوداء: `rgba(0,0,0,0.3)`
- توزيع عمودي للمحتوى
- محاذاة في المركز

**ج) الأيقونة الدائرية:**
```css
- الحجم: 50×50 px
- الخلفية: متدرجة (#8C0032 إلى #c41e5e)
- الشكل: دائري (border-radius: 50%)
- الظل: 0 2px 8px rgba(0,0,0,0.2)
- لون الأيقونة: أبيض
- حجم الأيقونة: 28×28 px
```

**د) نسبة الخصم:**
- الحجم: 18px
- الوزن: bold
- اللون: أبيض
- التأثير: text-shadow لوضوح أفضل

**هـ) اسم المميزة:**
- الحجم: 11px
- الوزن: 600 (semi-bold)
- اللون: أبيض
- المحاذاة: مركز
- الارتفاع: 1.2

### 2. شريط المميزات الإضافية

#### الخصائص:
- **التخطيط**: 3 أعمدة متساوية
- **الخلفية**: متدرجة (#8C0032 إلى #c41e5e)
- **الحشوة**: 12px
- **الفجوة**: 10px

#### المميزات:
1. **🔒 دفع آمن** (Shield Icon)
2. **🌍 تقبل عالمياً** (Globe Icon)
3. **🎁 مميزات حصرية** (Gift Icon)

#### أنماط النص:
- الحجم: 10px
- الوزن: 600
- اللون: أبيض
- المحاذاة: مركز

---

### 3. أشرطة المميزات الخاصة

#### الخصائص العامة:
- **الخلفية**: متدرجة (#8C0032 إلى #c41e5e)
- **اللون**: أبيض
- **الحشوة**: 12px
- **الهامش**: 12px من الجانبين
- **الحدود**: 10px border-radius
- **التخطيط**: flex (أيقونة + نص)

#### المميزات الثلاث:

**1. كاش باك:**
- الأيقونة: Wallet
- النص: "كاش باك حتى 5000 ريال شهرياً" / "Cashback Up to 5000 QAR Monthly"

**2. صالات VIP:**
- الأيقونة: Crown
- النص: "دخول صالات VIP في المطارات" / "VIP Lounge Access"

**3. جوائز سنوية:**
- الأيقونة: Trophy
- النص: "جوائز سنوية تصل إلى 3 مليون ريال" / "Annual Rewards Up to 3M QAR"

#### أنماط النص:
- الحجم: 12px
- الوزن: 600
- الفجوة من الأيقونة: 10px

## الأيقونات المستخدمة

جميع الأيقونات من مكتبة **Lucide React**:

```typescript
import {
  Car,           // تأجير السيارات
  Hotel,         // حجوزات الفنادق
  Plane,         // تذاكر الطيران
  ParkingCircle, // المواقف
  Shield,        // دفع آمن
  Globe,         // تقبل عالمياً
  Gift,          // مميزات حصرية
  Wallet,        // كاش باك
  Crown,         // صالات VIP
  Trophy         // جوائز سنوية
} from "lucide-react";
```
